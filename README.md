## Overview

This project demonstrates the implementation of **Zero Trust Network Access (ZTNA)** using **Zscaler Private Access (ZPA)** to secure user-to-application connectivity.
The Zero Trust model follows the principle **“Never Trust, Always Verify”**, ensuring that users can access only the applications they are authorized to use.

## Project Stages

### Stage 1 – Web Security Testing

Identified web application vulnerabilities based on **OWASP Top 10**.

### Stage 2 – Vulnerability Scanning

Performed vulnerability assessment using **Nessus Scanner**.

Target Website:
http://testfire.net/

### Stage 3 – ZTNA Implementation

Implemented **Zero Trust architecture** using **Zscaler Private Access (ZPA)** for secure application-level connectivity.

## Tools Used

* Nmap

* Nessus
* Metasploit Framework
* OWASP ZAP
* Burp Suite

## Key Features

* Identity-based access control
* Multi-Factor Authentication (MFA)
* Secure micro-tunnel connectivity
* Continuous monitoring and logging
* Reduced attack surface

## Conclusion

ZTNA improves cybersecurity by replacing traditional VPN models with **secure, identity-based access**, preventing unauthorized access and lateral movement within networks.
